# clothing-advisor

現在の気温・天気に応じて、当日の服装の厚さ指標と衣替え時期を提示するアプリ。

## 概要

性質の異なる2機能を同等に扱う。

- 当日の服装（連続・毎日利用）：予報から服装の厚さを昼／朝晩の二段で提示する
- 衣替え時期（離散・季節イベント）：週間平均気温の移動平均から入替え時期を判定する

## 主な機能

- 地点の気温（最高／最低）・天気・風速・湿度・7日予報の取得とキャッシュ
- 体感温度補正を経た服装レベルの提示（昼／朝晩の二段）
- 直近平均との相対提示（「一枚多め／少なめ」）
- ヒステリシス付きの衣替え判定
- 寒がり／暑がりの個人オフセット

## 技術構成

- Next.js
- Firebase（Firestore）
- 外部気象API（7日予報の提供が必須条件、下記参照）
